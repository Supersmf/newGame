About

Information about the Game

  • Genre

    War Space is a space combat arcade game, in which the player flies on a spaceship in space and fights off enemy ships

  • Technologies

    The game is built using: Webpack, JavaScript, Babel, ECMAScript 6, Canvas, Web Audio API

  • Resolution

    You can enjoy the game on almost all popular computers, with the exception of a touch screen device (this is an opportunity to appear in the following releases).

  • Difficulty

    This is not the most difficult game from which you could meet, but you can also find several kinds of enemies, as well as the main enemy in the form of a Death Star

How to win

When you reach 25000 points, you have the opportunity to fight with the main space enemy - the Death Star. For those who do not have the patience to earn the right amount of points, but have the desire to fight with the main enemy, can press the button "9" and get immediately 24900 points.
